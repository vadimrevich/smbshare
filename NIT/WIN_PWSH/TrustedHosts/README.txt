Set All Hosts Trusted

Данный скрипт делает все удалённые хосты в локальной сети и Интернете
доступными для небезопасного подключения по протоколу WinRM. Опция
предназначена для тестовой среды, поскольку позволяет устанавливать
соединение с передачей паролей в незашифрованном виде.

Для рабочей конфигурации (небезопасное подключение только внутри
локальной сети) подходит больше скрипт Set-LocalHostTrusted.ps1.

Данные скрипты следует использовать только на клиентских компьютерах. По
большому счёту на серверных операционных системах они бесполезны.
