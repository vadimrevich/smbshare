Adjust Server

Данный набор скриптов предлагается для первоначальной настройки
Microsoft Windows Server в организации, а именно:

-   Снятие режима усиленной безопасности Microsoft Internet Explorer
-   Отключение автозапуска Windows Server Manager при старте
    операционной системы
-   Отключение тригера сообщений при перезагрузке операционной системы
-   Отключение политики предотвращения слабых паролей на компьютерах, не
    входящих в домен;
-   Создание администратора с пользователем и пароле vagrant (для
    тестовых систем)
-   Подготовка кумулятивного обновления операционных систем Microsoft
    Windows
-   Отключение избыточной антивирусной защиты в Microsoft Windows
    Defender
-   Добавление обновлений системы для Exponenta System Update
-   И ещё много разных функций

## Распространение 

Набор скриптов распространяется в виде zip архива,
который вы можете скачать на локальный компьютер с веб-сервера. Далее вы
должны каким-то способом переслать его на Windows сервер. Поскольку при
первоначальной установке Windows Server блокирует доступ по
Интернет-протоколам, лучше всего это сделать при помощи общей папки,
доступной по протоколу SMB.
